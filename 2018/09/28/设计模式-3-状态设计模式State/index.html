在状态模式中可以有多个状态，同时对应多个分支程序;

状态模式和代理模式有相似之处，都有目标对象和代理的相似概念
但是他们之间是有区别的：
1.代理模式目标对象只有一个，而状态模式可以有多个目标对象，即，多个状态。
2.代理模式是为了控制目标对象的访问，状态模式是为了根据状态或者标示判断使用哪个目标。
